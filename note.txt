membuat module
    buat repo github
    buat module dengan go mod init (nama module)
    nama module menggunakan repo github, misal: github.com/LeaOhyeah/go-say-hello
    tanpa http nya

relase versi
    buat tag di git
    tag dimulai dengan huruf v
    misal v1.0.0


menambah dependensi (menggunakan module)
    dengan go get (nama module/pakcage lain)
    go get github.com/LeaOhyeah/go-say-hello
    sekarang di go.mod ada require baru, dan ada juga go.sum


upgrade module
    bikin tag baru aja

upgrade dependensi
    bisa ubah langsung isi go.mod
    lalu go get ulang semua module (go get aja)

major upgrade
    sangat dihindari jika tidak backward kompatible
    jika terpaksa, ganti nama module
    biasanya ditambah di akhir seperti
    github.com/LeaOhyeah/go-say-hello menjadi github.com/LeaOhyeah/go-say-hello/v2
    lalu dependensi berubah require nya
